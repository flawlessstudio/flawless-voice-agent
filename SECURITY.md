# Security Policy

## Reporting a vulnerability

Please do NOT open a public issue. Send a report to: oneflawlessstudio@gmail.com

Include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix if available

We will respond within 72 hours.

## Scope

- Voice session data
- PII and transcript handling
- API key management
- Compliance-related controls
- Infrastructure secrets
