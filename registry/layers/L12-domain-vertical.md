# L12 — Domain Vertical

## Core: Case-dependent

The domain vertical layer is selected per use case, not globally.

| Use case | Winner | Reason |
|---|---|---|
| Sales qualification | Thoughtly | Strong outbound sales flows, CRM native |
| Recruiting screening | Clara | Purpose-built for recruiting, strong compliance |
| Lead qualification | Vozy | Spanish/LATAM strong, multi-language |
| Support triage | Retell vertical | Flexible, strong inbound |

## Criteria for inclusion
- Must provide measurable improvement over generic agent for the specific vertical.
- Must not introduce compliance risk beyond what the core stack already handles.
